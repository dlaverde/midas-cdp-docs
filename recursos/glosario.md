# Glosario

Términos importantes del CDP de Midas.

## CDP
Customer Data Platform - Plataforma de datos de clientes.

## Composable CDP
CDP que usa tu data warehouse existente.

## Event
Acción rastreada (ej: "Product Viewed").

## Property
Atributo de un evento.

## Trait
Atributo de un usuario.

## Identity Resolution
Proceso de unificar perfiles de usuarios.

## Data Warehouse
Sistema de almacenamiento de datos (Snowflake, BigQuery, etc).

## Reverse ETL
Sincronización de datos del warehouse a herramientas operacionales.
