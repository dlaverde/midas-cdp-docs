# Data Warehouses

El CDP de Midas soporta los principales data warehouses.

## Warehouses Soportados

- [Snowflake](data-warehouses/snowflake.md)
- [Google BigQuery](data-warehouses/bigquery.md)
- [Amazon Redshift](data-warehouses/redshift.md)
- [ClickHouse](data-warehouses/clickhouse.md)
- [PostgreSQL](data-warehouses/postgresql.md)

## Configuración General

Todos los warehouses requieren:
- Credenciales de acceso
- Permisos de escritura
- Database y schema específicos

## Modos de Entrega

- **Stream**: Tiempo real, evento por evento
- **Batch**: Lotes periódicos (más eficiente)
