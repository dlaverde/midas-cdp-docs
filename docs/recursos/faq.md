# FAQ

## ¿Qué es el CDP de Midas?

Es una plataforma de datos de clientes que te permite capturar, almacenar y activar datos de eventos en tu propio data warehouse.

## ¿Cuál es la diferencia con Segment?

- ✅ **Tus datos viven en TU warehouse** (no en servidores de Segment)
- ✅ **Costos más bajos** (no pagas por eventos almacenados)
- ✅ **Compatible con Segment** (puedes migrar fácilmente)
- ✅ **Open-source** (transparencia total)

## ¿Dónde se almacenan mis datos?

En TU propio data warehouse:
- Snowflake
- BigQuery
- Redshift
- PostgreSQL
- ClickHouse

Tienes control total sobre tus datos.

## ¿Es compatible con Segment?

Sí, 100% compatible. Puedes:
- Usar las mismas librerías
- Migrar sin cambiar código
- Usar el mismo esquema de datos

## ¿Cómo obtengo mis credenciales?

Contacta a tu account manager para obtener:
- Write Key (para navegador)
- Server Secret (para backend)
- Tracking Host URL

## ¿Puedo trackear eventos desde mi backend?

Sí, usa:
- Node.js SDK
- HTTP API directa
- Cualquier lenguaje

## ¿Qué pasa si mi usuario tiene adblock?

El CDP usa técnicas para minimizar el impacto de adblockers. La mayoría de eventos llegarán correctamente.

## ¿Es GDPR compliant?

Sí. Como tus datos viven en TU warehouse, tienes control total para cumplir con GDPR, CCPA y otras regulaciones.
